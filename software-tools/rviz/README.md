# RViz

**RViz** (short for **ROS Visualization**) is the primary 3D visualization tool used in the Robot Operating System (ROS) ecosystem. Using various plug-ins, it's possible to visualize a robot's model, meshes, and the reference frames that make up the robot, facilitating the design and programming of multiple robots.

It also allows you to visualize information from different sensors such as cameras, lasers, LiDAR, Kinect devices, and others.

In simple words using RViz we can visualizate what the robot is "thinking" and "seeing" by pulling data from various sensors and software modules into a single 3D environment. Think of it as a **debugging window for the physical world**. While a simulator like Gazebo mimics physics and gravity, RViz simply displays the data that already exists in your ROS system.
